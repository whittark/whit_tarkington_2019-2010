API Projects

API calls with AJAX and JASON

The Utopia Intergalactic Travel site was a group project to practice collaborating check-ins within a GIT repository. We had fun building aspects of the site as a team. I provided the asteroid tracking page, which calls data from NASA's near-earth object database to map asteroids that may be hazardous during shuttle exits and entries.

The Ghibli GIF Gavotte is an app I built to practice using API calls with AJAX. The site uses JavaScript and jQuery to make the call and display animated GIFs based on themes users select.
