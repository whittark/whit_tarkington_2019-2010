XML Authoring

Developing interoperable content with DITA XML

By building content libraries in DITA XML, our writing teams ensure that our content can be pushed to responsive web or print transformations from a single content source.

Developing non-proprietary XML content also aligns us for the future. As tech comm solutions become more integrated within applications, we'll need files that can translate into a range of interface outputs, such as text within chat bots or guided interface tours. Open-source XML solutions afford us that interoperability.

Client: athenahealth
