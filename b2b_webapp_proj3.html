B2B Web Applications

In moving communications closer to the UI, there's no better way to integrate solutions then to learn how to build applications one's self.

LUXE LEISURE RESORT MANAGEMENT: I worked with two other developers to construct a business-to-business application for managing reservations within a resort.

Guests visit the site to create reservations which are posted to a MySQL database. Resort managers retrieve this data within an administrative backend to update the status of guests.

I developed the views and public files for the project, which include JavaScript to validate user entries in form fields.

Project Name: Luxe Leisure Resort Management Application
Skills: JavaScript, jQuery, NodeJS, Express, Sequelize, MySQL, Bootstrap, and Passport for user authentication
