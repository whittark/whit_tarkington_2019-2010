MySQL Projects


Posting and Pulling User Data

The MySQL database projects I developed shed light on methods I can use to collect and display data within communication solutions, such as a chat bot or guided interface tour.

In Dream Burger I practiced posting user data from a Bootstrap site to a MySQL database. The site pulls the stored data and displays it within the site with the option to perform user actions.

For the Scientific Explorer Matchup site, the online survey collects user data and retrieves the ideal lab partner based on user responses. The JavaScript used to evaluate a match is a step forward for matching user input with chat bot responses.
