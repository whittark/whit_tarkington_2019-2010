JavaScript and JQuery

UI Interaction with JavaScript and jQuery

I found the JavaScript and jQuery projects useful in planning the web elements to build for guided interface tours.

In Smegol's Barter, users must guess the number of gems to pay for a wish without knowing what each gem type is worth. Users win if they 'pay' the amount exactly or lose if they exceed the amount. JavaScript code evaluates the user's input; jQuery updates the elements and values displayed dynamically throughout the game.

For the Medieval Medical Trivia project, users guess at cures used for ailments in the middle ages. JavaScript code evaluates whether answers are correct; jQuery displays whether an answer is correct and reveals the final score


Skills: JavaScript, jQuery, Bootstrap, HTML5, and CSS4
